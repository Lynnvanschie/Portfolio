Inleiding Programmeren

Situatie

Ik heb deze opdracht in het eerste jaar van CMD uitgevoerd om het vak Inleiding Programmeren te halen. Hierbij moesten aantonen dat we de programmeer taal Javascript goed beheersen.

Taak

De opdracht was het maken van een applicatie in Javascript. Daarin moest je helemaal zelf een applicatie maken op basis van een aantal eisen. Deze opdracht was een individuele opdracht.

Activiteiten

De eerste paar weken van het blok heb ik gebruikt om de taal Javascript te leren kennen. Daarna heb ik research gedaan naar verschillende Javascript applicaties en heb ik een keuze gemaakt in wat ik wilde maken. De laatste twee weken ben ik aan de slag gegaan met de code die geschreven moest worden voor deze applicatie.

Resultaat

Uiteindelijk eindigde ik met een mooie piano die geluid kan maken door op de toetsen te klikken of door het toetsenbord te gebruiken. Ook kan je verschillende geluiden van andere instrumenten selecteren. Ik ben erg trots op het eindresultaat. Ik wilde ook nog graag een loop toevoegen om de geluiden herhaaldelijk af te spelen, maar dit is niet gelukt. De feedback die ik hierop kreeg was dat dit ook best ingewikkeld is en dat ik dat op dat moment nog niet kon.

Reflectie

Ik vind dit een goed voorbeeld voor verbeelden en maken, omdat ik een idee had om een piano te maken en dit ook werkelijk heb kunnen maken met Javascript. De interactie in het prototype werkt goed en ik vond het leuk om te expirementeren met de verschillende tonen.
